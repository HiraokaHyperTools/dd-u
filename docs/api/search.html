検索 API¶

URL¶

POST /DD/ando.php?a=search
QueryString¶

キー

値の設定

FindWay

FindWay の指定 を参照

q
省略可能
WCQ 検索式を BASE64 ('base64url' encoding) でエンコードしたもの
WCQ 検索式 を参照

userid
省略可能
未登録呼び出し対象の担当者 ID

start
省略可能
スキップ件数。0 の場合、最初から表示

FindWay の指定¶

FindWay の指定により、検索条件の設定方法が変化します。

FindWay 許容値¶

設定値

検索方法

kw
キーワード検索で検索
キーワード検索での条件指定 を参照

c
カスタム検索 1 で検索
カスタム検索 1 での条件指定 を参照

c2
カスタム検索 2 で検索

キーワード検索での条件指定¶

キーワード検索をしたい場合の指定です。

文字列項目の検索条件¶

詳細については オペコード を参照

condition 値¶

値

動作

s,contain

を含む 検索

s,notcontained

を含まない 検索

s,startswith

から始まる 検索

s,endswith

で終わる 検索

s,isempty

空白 検索

s,isnotempty

空白でない 検索

s,is

完全一致 検索

f,geq

以上の数値 検索

f,leq

以下の数値 検索

日付項目の検索条件¶

詳細については オペコード を参照

condition 値¶

値

動作

d,is

と等しい日付 検索

d,between

右の日付けまで 検索

d,leq

よりも前の日付 検索

d,geq

よりも後の日付 検索

d,isnot

以外の日付 検索
