HTTP レスポンスボディー¶

JSON 交じりの HTML で出力します。

<html><body><!-- <mydd:ando>XXX</mydd:ando> --></body></html>

XXX の部分に JSON が入ります。

JSON の構成:

{
   "feat": {
      "mydd:ando":1,
      "0.1":1,
      ...
   },
   "PRO4_CORP_NAME": "",
   "PRO4_VER": "4.2.xxx"
}

feat の概要¶

  • デジタルドルフィンズのバージョンアップにより、機能が増えていきます。

  • "mydd:ando":1 の場合は、mydd:ando を備えていることを意味します。

  • "mydd:ando":0 は使用しません。項目そのものが無くなります。
